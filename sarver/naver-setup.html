네이버 오류 207 해결

disp_stat=207 → Callback URL이 개발자 센터 등록값과 다릅니다.

아래 값을 그대로 네이버 개발자 센터에 등록하세요.

1. PC 웹 서비스 URL

__SERVICE_URL__

2. PC Callback URL (최대 5개 중 1번에 등록)

__CALLBACK_URL__

등록 경로

  1. developers.naver.com/apps 접속
  2. 애플리케이션(MyApp) 선택
  3. API 설정 탭
  4. 네이버 로그인 → 서비스 URL / Callback URL 입력
  5. 저장 후 1~2분 기다린 뒤 다시 로그인

참고: localhost와 127.0.0.1은 다른 주소입니다.
